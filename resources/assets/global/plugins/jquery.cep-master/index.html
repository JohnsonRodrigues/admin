Como Funciona

jQuery CEP Plugin utiliza um webservice de CEP para pegar dados do endereço de acordo com o CEP.

Utilização Básica

O jeito mais simples de utilizar o plugin é criar um formulário de endereço e, em cada campo, colocar os atributos requeridos.

Callback

O callback é uma função chamada toda vez que um CEP é preenchido. Esta função recebe um argumento, que são os dados do endereço.

  • Método 1
  • Método 2
  • Exemplo de Resposta
$('#cep').cep(function(endereco) {
    // Veja no console do seu navegador
    // toda vez que terminar de preencher um CEP
    console.log('O endereço encontrado foi: ', endereco);
    
    // Seja criativo, você pode salvar no banco de dados
    // ou preencher outros campos
    // ou qualquer outra coisa que queira fazer
});

$('#cep').cep({
    // "done" é o nome da opção do seu callback
    // Funciona da mesma forma que o método 1
    // mas aqui você pode adicionar outras opções também
    done: function(endereco) {
        console.log('O endereço encontrado foi: ', endereco);
    },
    
    // Outras opções, caso você queira
    autofill: false,
    autofill_attr: "data-cep"
});

Preenchimento Automático

Após usar o jQuery CEP Plugin em um campo de texto, você pode configurar o preenchimento automático de outros campos simplesmente com alguns atributos data.

Sempre que o campo CEP é preenchido, automaticamente os campos com atributos são preenchidos, e você não precisa se preocupar em atribuir os valores manualmente para cada campo do endereço.

<input type="text" ... data-cep="cidade"> Vai ser preenchido automaticamente com a cidade do CEP informado
<input type="text" ... data-cep="uf"> Vai ser preenchido automaticamente com o estado (UF) do CEP informado
<input type="text" ... data-cep="tipo_logradouro"> Vai ser preenchido automaticamente com o tipo de logradouro do CEP informado (rua, avenida, travessa, beco, etc...)
<input type="text" ... data-cep="logradouro"> Vai ser preenchido automaticamente com o logradouro do CEP informado (o nome da rua, ou da avenida, etc...)
<input type="text" ... data-cep="endereco"> Vai ser preenchido automaticamente com o tipo de logradouro + logradouro do CEP informado (Rua Shake, Avenida Brasil, etc...)
<input type="text" ... data-cep="bairro"> Vai ser preenchido automaticamente com o bairro do CEP informado
Você pode personalizar o uso do plugin utilizando as opções disponíveis listadas abaixo.
Opções do Plugin
function
done

default: function() {}

Callback que recebe um argumento com os dados do CEP.
Esta função é executada sempre que o campo CEP for preenchido.

boolean
autofill

default: true

Quando o campo CEP é preenchido, além de chamar o seu callback existe esta opção para que os demais campos do formulário se preencham automaticamente.

Para que o preenchimento seja feito corretamente, você deve pôr um determinado argumento nos campos que você quer que sejam preenchidos automaticamente, o argumento é definido com a opção autofill_attr.

Exemplo:

<input type="text" data-cep="rua"> - Será preenchido com o campo "rua"

string
autofill_attr

default: "data-cep"

A opção de preencher dados automaticamente (autofill) vai preencher apenas os inputs que tenham um determinado argumento, esta opção define o nome deste argumento.
